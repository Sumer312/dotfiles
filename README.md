# My Setup
* OS: Debian
* Window Manager: DWM
* Compositer: picom
* Shell: Bash
* bin/sh: dash
* Prompt: 
    - bash: starship
    - zsh: powerlevel10k
* Terminal: Alacritty
* Application Launcher: Rofi
* Editor: Neovim
* File Manager: Vifm
* Notification Daemon: Dunst
* Image Viewer: feh
* Video Player: mpv
* PDF/EPUB: zathura
* Lock: i3lock
